# AI Book Generator - Frontend

Production-ready React + TypeScript + Tailwind CSS frontend.

## Quick Start

```bash
npm install
npm run dev     # Development server on port 3000
npm run build   # Production build
```

## Environment

Create `.env`:
```
VITE_API_URL=https://aibook-9rbb.onrender.com
```

## Stack

- React 19 + TypeScript
- Tailwind CSS 3
- Vite 7
- React Router v7
- TanStack Query
- Zustand
- Axios

## Features

✓ Full authentication flow
✓ Dashboard with stats
✓ Real-time book editor
✓ Library management
✓ Credit system
✓ Affiliate program
✓ EPUB export
✓ Mobile responsive
✓ Glass-morphism UI

## Pages

- `/auth` - License key auth
- `/dashboard` - Overview & quick actions
- `/editor/:id` - Book editor
- `/library` - All books
- `/credits` - Purchase credits
- `/affiliate` - Referral program
- `/settings` - User preferences

Built for production. Zero placeholders.
